  • MIPS Processor (expand)

    I designed and tested a processor in a team of four. We developed an architecture which used load-store and a number of registers. We also designed an instruction set for the processor and wrote a compiler to convert the assembly code into binary. At the end of the project, the processor was able to run a program to find the GCD of two numbers using Euclid's Algorithm.
